# 第四章 对象和类
## 4.1 面向对象程序设计概述
- **OOP** 把数据结构放第一位，算法放第二位
### 4.1.1 类
- 概念
	- 类 **class**
	- 构造 **construct**
	- 实例 **instance**
	- 封装 **encapsulation**
	- 实例域 **instance field**
	- 方法 **method**
	- 状态 **state**
	- 继承 **inheritance**
### 4.1.2 对象
- 特性
	- 行为 **behavior**
	- 状态 **state**
	- 标示 **indentity**
