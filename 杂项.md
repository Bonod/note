```c
#define offsetof(TYPE,MEMBER)   (size_t) &((TYPE *)0)->MEMBER)
```
2. 计算字节中`bit`为`1`的数量
``` c
int Count(BYTE v)
{
    int num = 0;
    while (v)
    {
        v &= (v - 1); //v=v&(v-1)这个操作可以直接消除掉v中的最右边的1。
        num++;
    }
    return num;
}
```
